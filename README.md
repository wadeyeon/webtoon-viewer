# Webtoon Viewer

사용자가 이미지를 업로드하면 이미지를 보여주는 웹툰 뷰어 사이트

## Description:

Vite로 프로젝트 생성하고, Material UI 오픈소스를 이용하여 작업하였습니다.

## Build with:

- ReactJS
- Material UI

## Features:

- 업로드한 이미지 보여주기
- 모달창 구현
- 다크모드 구현
- 이미지 업로드 구현
- back to top 버튼
- fab(floating action button)
