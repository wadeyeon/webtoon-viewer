# Webtoon Viewer

Youtube OPEN API를 이용한 Youtube 클론 코딩

# color

## Description:

vite로 프로젝트 생성

## Build with:

- ReactJS
- material

## Features:

- 사용자가 선택한 이미지 순서대로 보여주기 (\_001)
- 이미지 hover 했을 때 파일명 보여주기
- column 수(4) 추가 및 삭제
- 각 영역별 스크롤
- 다크 모드 구현
- top/down 버튼
- 이미지 구분선 toggle
- 영역 스크롤 잠금 toggle
- 드래그앤드랍

- 뷰어 (모바일, 웹)
